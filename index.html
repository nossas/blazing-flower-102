Bem-vindo!

O MeuRio é um movimento democrático e apartidário, que usa ferramentas on-line para conectar os cidadãos do Rio de Janeiro (Brasil) e garantir que sua voz oriente os processos de decisão que estão transformando a cidade.

Sobre este repositório

Nós utilizamos Ruby on Rails para construir nossas ferramentas (que note: são totalmente opensource) e se você quiser colaborar com o projeto (http://meurio.org.br), sinta-se a vontade!

Iniciando o servidor do projeto

Se você já tem experiência com Ruby on Rails, dê "fork" no projeto e siga as seguintes instruções:

$ cd sua_pasta_de_projetos_ou/
$ git clone git@github.com:SEU_USUARIO/meurio.git
$ cd meurio/
$ rake db:create db:migrate db:test:prepare
$ -- suas alterações --
$ git add .
$ git commit -m "Sua mensagem sobre a alteração"
$ git push

No seu fork, faça uma pull request Se precisar de dados de exemplo, acrescente rake db:seed após rodar o comando rake acima.
